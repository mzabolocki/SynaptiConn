DATASET INFO
------------

The all_unit.mat file contains spike times published in the following paper: https://star-protocols.cell.com/protocols/3438#bib12

These are for tetrode recordings, taken in the hippocampus of rats. The data is from the Balbi Lab, QBI (Queensland Brain Institute, Australia).

Data was spike-sorted in MATLAB using Wave_Clus.

See the additional paper for further reports: https://www.cell.com/cell-reports/fulltext/S2211-1247(23)01487-0?_returnURL=https%3A%2F%2Flinkinghub.elsevier.com%2Fretrieve%2Fpii%2FS2211124723014870%3Fshowall%3Dtrue
